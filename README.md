# 🚀 AI Engineering Studio

> **An AI-powered GitHub Repository Analysis Platform for Developers**

AI Engineering Studio is a full-stack platform that intelligently analyzes GitHub repositories and provides deep insights into project architecture, code quality, dependencies, security, and overall repository health.

Designed for developers, recruiters, and engineering teams, the platform transforms raw repositories into actionable engineering reports using AI-powered analysis.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/Yogesh-kk49/AI-Engineering-Studio.git
cd AI-Engineering-Studio
```

## Backend

```bash
cd backend

python -m venv venv

# Windows
venv\Scripts\activate

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```

---

## Frontend

```bash
cd frontend

npm install

npm run dev
```

---

## Docker

```bash
docker compose up --build
```

# 🚀 How It Works

1. User enters a GitHub repository URL.
2. The platform retrieves repository metadata.
3. Repository files and structure are analyzed.
4. Languages, frameworks, and dependencies are detected.
5. AI generates engineering insights.
6. Results are displayed through an interactive dashboard.
